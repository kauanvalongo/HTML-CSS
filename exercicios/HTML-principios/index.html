Negrito / Itálico em html:

Negrito:

Podemos escrever um texto em negrito utilizando a tag [ b ] (não semântico)

Ou também escrever um Termo Destacado utilizando a tag [ STRONG ] (semântico)

Itálico;

Nesta Frase utilizamos um termo itálico utilizando a tag [i](não semântico)

Nesta frase temos um Termo Ênfase utilizando a tag [EM](semântico)
